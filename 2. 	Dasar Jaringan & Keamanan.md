## 4. Dasar HTTP/HTTPS & REST API
- Cek header website:
  ```
  curl -I https://google.com
  ```

  <img width="1819" height="298" alt="image" src="https://github.com/user-attachments/assets/0b129a92-35f6-4a44-b169-e8b71e80176f" />

- Bandingkan HTTP vs HTTPS:
  ```
  curl -I http://example.com  # Tanpa enkripsi
  curl -I https://example.com # Dengan enkripsi
  ```

  <img width="1840" height="550" alt="image" src="https://github.com/user-attachments/assets/44828550-80ae-4522-a59b-655ea92f77a2" />

  http://google.com = Protokol HTTP, tidak terenkripsi.
  https://google.com = Protokol HTTPS, terenkripsi via TLS/SSL.

- Lihat sertifikat SSL:
  ```
  openssl s_client -connect google.com:443 | openssl x509 -noout -text
  ```
